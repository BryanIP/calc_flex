# Calculadora Flex - Combustível ⛽🚗🏍️

Este é o projeto prático do curso [7192 - Flutter Apps: Álcool ou Gasolina](https://balta.io/cursos/flutter-apps-alcool-ou-gasolina) do [balta.io](https://balta.io/).
